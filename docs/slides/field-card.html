Fit Guide
When to Deploy This
Good fit
Customer wants to see an agent built end-to-end
Technical team present and wants keyboard time
Sales stage U2/U3 — use case defined or being scoped
Team wants to understand agent eval and hardening
First exposure to Databricks Apps, Lakebase, or MLflow evals
Not the right fit
No technical team in the room
Customer still exploring use cases → Agent Bricks instead
Need customer's own data same day → plan custom path ahead
Less than 90 minutes available
